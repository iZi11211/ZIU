# React Todo + Register App

## 📌 Opis projektu
Aplikacja typu SPA napisana w React, zawierająca:
- system zarządzania zadaniami (Todo)
- wieloetapowy formularz rejestracji
- mock API (MSW)
- global state management
- animacje UI (Framer Motion)
- routing między widokami

Projekt został przygotowany w ramach zajęć laboratoryjnych.
